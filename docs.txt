Что такое Структуры данных?
Структуры данных -  и есть структуры, которые могут хранить типы данные вместе, они используются для хранения связанных данных.

Зачем они нужны?
Структуры данных служат для написание программ,которые могут быть так и простыми,так и достаточно сложными.

Что такое Hash Map?
Hash Map - это структура данных, которая реализует абстрактный тип данных ассоциативного массива с использованием ключей и значений.

Что такое Linked List?
Linked List - это структура данных, вотличие от обычного list, linked list оно сохроняет разбросонно и показывает на предедущий и на следующий индекс.

В каких случаях использовать Linked List и Hash Map?

Hash Map - чтобы сделать минимум опираций то есть Big O будет состовлять 1
Linked List - нужен для того чтобы ОЗУ использовалась на 100% , и чтобы не было нагрузка на систему
